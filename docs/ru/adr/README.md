# Записи об архитектурных решениях

Этот каталог содержит записи о значимых архитектурных решениях.

Используйте последовательную нумерацию имен файлов, например `0001-short-decision-title.md`. Каждый ADR должен содержать разделы:

- Статус
- Контекст
- Решение
- Последствия

Принятые ADR являются неизменяемыми историческими записями. Если решение меняется, добавьте новый ADR, который заменяет предыдущий, и обновите внутренний перечень решений.

- [`0002-configuration-dsl.md`](0002-configuration-dsl.md) — ConfigurationVersion как декларативный Configuration DSL, исполняемый Runtime.
