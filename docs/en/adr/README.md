# Architecture Decision Records

This directory contains records of consequential architecture decisions.

Use sequential filenames such as `0001-short-decision-title.md`. Each ADR should include:

- Status
- Context
- Decision
- Consequences

Accepted ADRs are immutable historical records. If a decision changes, add a new ADR that supersedes the earlier record and update the internal decision index.
